# Shelby Media Vault

Shelby Media Vault is a simple web app that allows users to upload and manage media files using Shelby storage.

## Features
- Upload media files
- Preview uploaded files
- Organize files in a vault-style interface

## Built With
- Next.js
- React
- TypeScript
- TailwindCSS

## Status
Work in progress. More Shelby storage features will be added.
